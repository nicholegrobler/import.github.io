Jira TM

This page will be showing how github links to jira task manager. Due to the tasks changing quickly and daily, downloading/exporting the task would be - in my opinion - a not such a good idea as every time the ticket is updated, the export/download would need to be redone.

I also think, every user in our team who has jira (i will be using Maribel, Luis, Jose and I for demonstration) should have their own sector in the page which gives easy access.

This can be used as a seporator or to add more information. In this use case i used it to link to the sprint as a whole.
